Integração Salesforce
Como configurar
1

Criar Connected App no Salesforce

Acesse Setup → App Manager → New Connected App. Ative OAuth, adicione os escopos api e refresh_token. Callback URL: https://localhost

2

Copiar Consumer Key

Na Connected App criada, clique em Manage Consumer Details e copie o Consumer Key.

3

Obter Security Token

Perfil → Settings → Reset My Security Token. Chegará por e-mail. A senha a informar abaixo é: SuaSenha + Token (sem espaço).

4

Preencher e conectar

Informe os dados abaixo e clique em Conectar.
